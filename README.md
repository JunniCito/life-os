# ⚡ Life OS - Sistema Operativo Personal

¡Bienvenido a **Life OS**! Un sistema de productividad y gamificación personal interactivo diseñado para ayudarte a gestionar tus hábitos, finanzas, entrenamiento físico, tareas académicas y bienestar emocional en un solo lugar. 

Este proyecto está construido con **React** y **Vite**, ofreciendo una interfaz súper rápida, fluida y con una estética futurista inspirada en los juegos RPG.

---

## 🚀 Características Principales

*   **⚡ Dashboard General**: Visualiza un resumen rápido de tu día, nivel RPG actual, frase motivacional diaria y barra de progreso general.
*   **⚖️ Módulo Fitness**: Registra tu peso, consumo de calorías, agua, pasos diarios y horas de sueño. Incluye cálculo de IMC y gráficos interactivos.
*   **💰 Control de Finanzas**: Gestiona ingresos y gastos, define metas de ahorro semanal y anual, y analiza tus gastos por categorías.
*   **🔥 Hábitos Checklist**: Controla tus rutinas diarias con un mapa de calor visual de consistencia semanal y estadísticas por hábito.
*   **🎓 Agenda Académica**: Añade tareas, proyectos y exámenes. Define prioridades y gana **+40 XP** al completarlas.
*   **👑 Sistema de RPG & Niveles**: ¡Sube de nivel a medida que ganas XP completando hábitos y tareas! Desbloquea logros y escala por los títulos de Novato a Leyenda.
*   **💑 Modo Pareja**: Registra hábitos y actividades compartidas para crecer juntos.
*   **📊 Revisión & Emociones**: Registra tu estado de ánimo diario, niveles de energía y haz reflexiones semanales/mensuales de tu rendimiento.

---

## 🛠️ Tecnologías Utilizadas

*   **React** (Componentes reactivos y Hooks avanzados para gestión de estado local persistente)
*   **Vite** (Empaquetador ultra-rápido para el desarrollo moderno de aplicaciones web)
*   **CSS en JS & Inline Styles**: Diseño personalizado con una paleta oscura sofisticada, gradientes modernos y transiciones fluidas.
*   **SVG Dynamic Charts**: Minigráficos vectoriales fluidos sin dependencias externas pesadas.
*   **LocalStorage**: Persistencia automática de todos tus datos para que no pierdas tu progreso al recargar la página.

## 💻 Cómo Iniciar el Proyecto Localmente

Para instalar las dependencias e iniciar el servidor de desarrollo local, sigue estos pasos:

1.  **Clonar el repositorio o descargar el proyecto**:
    ```bash
    git clone <tu-url-de-github>
    cd lifeos-dashboard
    ```

2.  **Instalar las dependencias**:
    ```bash
    npm install
    ```

3.  **Iniciar el servidor en modo de desarrollo**:
    ```bash
    npm run dev
    ```

4.  Abre tu navegador en la dirección local que se muestre (normalmente `http://localhost:5173`).

---

## 📦 Construcción para Producción

Para compilar y optimizar la aplicación para producción:

```bash
npm run build
```

Esto generará los archivos finales optimizados en la carpeta `dist/`, listos para subir a cualquier hosting gratuito (como Vercel, Netlify o GitHub Pages).
